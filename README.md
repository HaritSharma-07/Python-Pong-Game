# Pong Game

A classic Pong Game built using Python and the Turtle graphics module. This project is part of my Python learning journey and helped me understand object-oriented programming, collision detection, game logic, and animation.

## Features

- Two-player gameplay.
- Paddle movement using keyboard controls.
- Ball collision with paddles and walls.
- Score tracking for both players.
- Ball speed increases after each successful hit.

## Technologies Used

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/Python-Pong-Game.git
```

2. Open the project folder.

3. Run the following command.

```bash
python main.py
```

## Controls

| Player | Keys |
|---------|------|
| Left Paddle | W / S |
| Right Paddle | Up / Down Arrow |
